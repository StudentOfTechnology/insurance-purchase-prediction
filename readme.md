# Insurance Purchase Prediction

This project uses machine learning to predict whether a customer will purchase vehicle insurance. The goal is to help insurance companies identify high-probability customers for targeted marketing campaigns.

## Project Overview

Insurance companies often have large customer bases and limited marketing resources. By predicting which customers are most likely to purchase insurance, companies can focus their outreach efforts more effectively.

This project builds a machine learning model to predict customer interest in vehicle insurance and visualizes the insights using Power BI.

## Tools Used

Python  
Pandas  
Scikit-learn  
Jupyter Notebook  
Power BI  

## Machine Learning Model

Random Forest Classifier

Model performance:

AUC Score: 0.86

## Key Predictors Identified

Vehicle damage history  
Previous insurance status  
Vehicle age  

## Key Insights

Customers with vehicles older than two years have the highest predicted probability of purchasing insurance.

Customers with previous vehicle damage are significantly more likely to purchase insurance coverage.

These insights can help insurance companies target marketing campaigns toward the most promising customer segments.

## Dashboard

The Power BI dashboard includes:

• Insurance purchase rate KPI  
• Vehicle age vs purchase rate analysis  
• Predicted purchase probability by vehicle age  

The dashboard demonstrates both historical analytics and predictive insights from the machine learning model.
